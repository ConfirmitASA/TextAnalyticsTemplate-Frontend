new AggregatedTable(table, hierarchy, fixedHeader, sorting)

Source:
  • aggregatedTable/AggregatedTable.js, line 17

A class that unifies work with Aggregated tables. Enabling a fixed header, search functionality, sorting, hierarchy etc can be done here.

Parameters:
Name Type Description
table HTMLTableElement

table on which the action is to be performed
hierarchy Object

config for HierarchyTable. See HierarchyTable docs for available object properties. If hierarchy.source is not defined, it will automatically receive value from table
fixedHeader Object

config for FixedHeader. See FixedHeader docs for available object properties. If fixedHeader.source is not defined, it will automatically receive value from table
sorting Object

config for SortTable. See SortTable docs for available object properties. If sorting.source is not defined, it will automatically receive value from table. If sorting.data is not explicitly defined, it will automatically receive value from hierarchy.data

This function adds toggle buttons for hierarchy (to toggle the flat-view/tree-view mode) to the hierarchy column (here available as this.hierarchy.column). This function is executed for both the original table and the cloned header.

Parameters:
Name Type Attributes Description
host HTMLTableCellElement

reference to the cell which acts as a header for hierarchy column
buttonClassChunk String

the part of the icon-class name that follows the .icon-. Is used to reference the icon class, but also used for the button without the .icon- prefix
flat Boolean

Boolean to be set on hierarchy.flat when this button is clicked.
title String <optional>

the title that describes what the button does to be show on hover (is set to native attribute title on the button element)
